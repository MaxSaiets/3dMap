# API для отримання даних рельєфу (висот)

## Безкоштовні опції (без API ключа)

### 1. OpenTopography (рекомендовано)
- **URL**: https://opentopography.org/
- **Дані**: SRTM, NASADEM, ASTER GDEM
- **Обмеження**: Безкоштовно для некомерційного використання
- **API ключ**: Не потрібен для базового використання
- **Якість**: Добра (30-90м роздільність)

### 2. USGS 3D Elevation Program
- **URL**: https://www.usgs.gov/3d-elevation-program
- **Дані**: LiDAR для США
- **API ключ**: Не потрібен
- **Обмеження**: Тільки для США

## Платні опції (потрібен API ключ)

### 1. Mapbox Terrain API
- **URL**: https://www.mapbox.com/
- **API ключ**: Потрібен (безкоштовний tier: 50,000 запитів/місяць)
- **Якість**: Висока
- **Встановлення**:
  ```bash
  # Додайте в .env файл
  MAPBOX_API_KEY=your_api_key_here
  ```

### 2. Google Elevation API
- **URL**: https://developers.google.com/maps/documentation/elevation
- **API ключ**: Потрібен (безкоштовний tier: 25,000 запитів/день)
- **Якість**: Дуже висока
- **Встановлення**:
  ```bash
  # Додайте в .env файл
  GOOGLE_ELEVATION_API_KEY=your_api_key_here
  ```

### 3. OpenElevation API
- **URL**: https://open-elevation.com/
- **API ключ**: Не потрібен (але є обмеження)
- **Якість**: Середня
- **Обмеження**: Рекомендовано не більше 100 точок на запит

## Поточна реалізація

За замовчуванням використовується **синтетичний рельєф** (хвильовий паттерн) для демонстрації.

Для реального рельєфу:
1. Встановіть API ключ (якщо потрібен)
2. Додайте в `.env` файл
3. Оновіть код в `elevation_api.py` для використання API

## Рекомендації

- **Для тестування**: Використовуйте синтетичний рельєф (вже реалізовано)
- **Для продакшн**: 
  - Mapbox (якщо є бюджет)
  - OpenTopography (якщо безкоштовно)
  - Локальні DEM файли (найкраща якість, але потрібно завантажувати)

## Приклад використання Mapbox

1. Зареєструйтеся на https://www.mapbox.com/
2. Отримайте API ключ
3. Додайте в `.env`:
   ```
   MAPBOX_API_KEY=pk.eyJ1IjoieW91cnVzZXJuYW1lIiwiYSI6ImN...
   ```
4. Код автоматично використає Mapbox API
